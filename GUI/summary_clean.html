Have you already under taken quality control (QC) on your Genetic data? Certain QC procedures like strand flipping require validation data so if your genetic data has not been QCed you should undertake that process first.

No i have not QC'ed my genetic data

Work with genetic data is often done on a super computer, so variables are not validated in this step. However, this pipeline will generate a validation file that can be run before submission to check that you have set everything correctly before submitting work to a que.

Genetic Path

If you have genetic data split by chromosome, please provide a path to that directory, otherwise please specify the path including the prefix if it is a plink file, or a full path to a bgen file

For example I:\Work\UKBiobankQCed, where this contains chr1.bed / chr2.bgen etc for a path

Or I:\Work\UKBiobankQced\biobank, for plink files you don't need to add the .bed but it will not fail if you do. Where as if you have a bgen file you must add the .bgen
